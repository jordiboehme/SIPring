About SIPring

SIP Phone Ringing Service

SIPring

Version {{ version }}

A Docker-based SIP phone ringing service for triggering alerts via HTTP requests. Perfect for doorbell notifications, alarm systems, and home automation.

Features
HTTP Triggers
Ring any SIP phone via simple HTTP GET requests
Custom Caller ID
Display custom names on the phone
Multiple Configs
Create different ring configurations for various use cases
Event Log
Track every ring with source, duration, and result history
Docker Ready
Easy deployment with Docker and docker-compose
